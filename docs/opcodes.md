# Opcodes

| name | opcode         | arguments |
|------|----------------|-----------|
| nop  | ```00000000``` |           |
| jmp  | ```10000001``` | address   |
| je   | ```10000010``` | address   |
| jne  | ```10000011``` | address   |
| ldr  | ```100100yy``` | address   |
| wrr  | ```100101xx``` | address   |
| mov  | ```0010xxyy``` |           |
| add  | ```0011xxyy``` |           |
| sub  | ```0100xxyy``` |           |
| cmp  | ```0101xxyy``` |           |
